How It Works

Public Site (index_mimo_code4.html)
Visitors see the beautiful Suphancasting site with 11 languages
AI Agents section shows status cards with live heartbeat
No controls visible — agents work silently in background

Admin Dashboard (admin.html)
Login screen → full agent control center
6 agents: coordinator, researcher, writer, visual, seo_optimizer, publisher
Launch Agents — runs full pipeline (research → blog → image → video → SEO → publish)
Self-Cycle — auto-generates random goals and runs autonomously
Auto-cycle — runs every N minutes automatically
Task DAG — live dependency graph visualization
Content tabs — blog, image prompts, video scripts, SEO reports
SEO scoring — 12-point checklist with ring score
Live log — real-time agent activity feed
Token tracking — sparkline usage trend
