🏜️ คืออะไร / What Is It
การหล่อทราย คือวิธีผลิตชิ้นงานโลหะโดยเทโลหะหลอมเหลวลงในแม่พิมพ์ทราย เป็นกระบวนการที่นิยมที่สุดในโลก คิดเป็น70%ของงานหล่อทั้งหมด

Sand casting creates metal parts by pouring molten metal into sand molds — the world's most used casting method, ideal for complex custom shapes.
⚗️ วัสดุ / Materials
• ทรายซิลิกา — ทนความร้อนสูง
• ดินเหนียว/เบนโทไนต์ — สารยึดเกาะ
• เหล็กหล่อ/เหล็กกล้า — โลหะหลัก
• Silica sand — heat-resistant mold base
• Clay binders — hold sand shape
• Cast iron/steel — poured metal
✅ ข้อดี / Advantages
✓ รูปทรงซับซ้อน / Complex shapes
✓ ต้นทุนแม่พิมพ์ต่ำ / Low tooling cost
✓ ชิ้นงานทุกขนาด / Any batch size
✓ หลากหลายโลหะ / Any metal type
✓ งานที่ทนความร้อน / High-temp parts
⚙️ ขั้นตอนการผลิต · Production

Production
Process

ขั้นตอน 1–3 / Steps 1–3
1
เตรียมแบบหล่อ / Pattern Making
สร้างแบบจำลองจากไม้หรือโลหะ / Create part pattern from wood or metal
2
ทำแม่พิมพ์ทราย / Mold Making
อัดทรายรอบแบบหล่อให้แน่น / Pack sand tightly around the pattern
3
ทำแกนทราย / Core Making
สร้างแกนสำหรับช่องว่างภายใน / Sand cores for internal cavities
ขั้นตอน 4–6 / Steps 4–6
4
หลอมและเทโลหะ / Melt & Pour
หลอมโลหะ >1300°C แล้วเทลงแม่พิมพ์ / Melt >1300°C, pour into mold
5
ถอดแม่พิมพ์ / Shakeout
ทุบแม่พิมพ์ทรายออกหลังแข็งตัว / Break sand away after solidification
6
ตกแต่งและตรวจสอบ / Finish & Inspect
เจียร ตัดชิ้นส่วนเกิน ตรวจสอบคุณภาพ / Grind, cut gates, quality check
